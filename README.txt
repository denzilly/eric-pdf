######## Eric-PDF V1.0 Readme #########

This script can be used to extract site data (unitID, productID, and serial number) from Ericsson PDF site files.

How to operate:

1. Make sure that all site files are placed in the /eric-pdf/data folder
2. in command prompt, run "python setup.py install --user" to install required dependencies
3. cd to the eric-pdf folder
4. run the pdf-scraper by entering the command "python pdf-scraper.py"
